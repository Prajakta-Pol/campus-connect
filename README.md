# Campus Connect – Smart Student Growth Platform

## 📌 Overview

Campus Connect is a comprehensive mobile application designed to digitally transform a student’s entire college journey.

It acts as a **mini LinkedIn for campus**, enabling students to:

* Build verified professional profiles
* Track academic and extracurricular growth
* Automatically generate resumes
* Collaborate with peers
* Engage with events and opportunities

The platform also replaces traditional **mentor files** with a smart, digital system.

---

## 🎯 Problem Statement

Students in colleges face several challenges:

* Lack of a centralized platform for achievements and skills
* Manual and outdated mentor tracking systems
* Difficulty in building authentic resumes
* Limited visibility of opportunities and events
* Struggles in finding suitable teammates for projects

This results in poor career readiness and missed opportunities.

---

## 💡 Proposed Solution

Campus Connect solves these challenges by providing:

* A unified student profile system
* Automated resume generation based on verified data
* Event tracking and participation logging
* Peer collaboration platform
* Digital mentor interaction system

## ▶️ How to Run the Project

### 1️⃣ Clone Repository

```
git clone https://github.com/your-username/campus-connect
cd campus-connect
```

---

### 2️⃣ Backend Setup

```
cd backend
npm install
npm run dev
```

---

### 3️⃣ Mobile App Setup

```
cd mobile-app
npm install
npx expo start
```

---

### 4️⃣ Database Setup

* Install PostgreSQL
* Create database:

```
CREATE DATABASE campus_connect;
```

* Run schema scripts from `/database`
